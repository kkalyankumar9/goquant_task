## Features

3D Globe with interactive rotation, zoom, and pan.

Exchange Server Markers with provider color coding (AWS, GCP, Azure).

Real-Time Latency Visualization with animated arcs and color-coded ranges.

Historical Latency Trends with min, max, and average stats.

Filters & Controls: toggle providers, select exchanges, filter latency range.

Fully responsive and mobile-friendly.

## Tech Stack

Next.js 16 + TypeScript

React + Three.js + react-globe.gl

Recharts for charts

TailwindCSS for styling

React Context + useReducer for state management

## Application start locally
### Install dependencies
npm install
### Run the app
npm run dev
